Machine Learning Model Evaluation and Debugging

Advanced Level 1.5 Hours

Course Overview

Learn advanced techniques for evaluating and debugging machine learning models using AWS services like Amazon SageMaker Clarify and Debugger.

Prerequisites

  • 1+ year experience with SageMaker and AWS ML services
  • 1+ year in related role (backend developer, DevOps, data engineer)
  • Python programming fundamentals
  • Prior courses in AWS ML Engineer Associate Learning Plan

Learning Objectives

  • Create and analyze model performance baselines
  • Evaluate classification and regression metrics
  • Identify and resolve model convergence issues
  • Use SageMaker Clarify for model interpretability
  • Conduct reproducible ML experiments
  • Debug models with SageMaker Debugger

    Understanding model transparency:

    SageMaker Clarify helps you understand and explain your ML models by providing:

    • Bias detection

      • Demographic disparity metrics
      • Bias mitigation techniques
    • Feature importance
      • SHAP (SHapley Additive exPlanations) values
      • Global feature importance rankings
      • Local feature attributions
      • Feature correlation analysis
    • Model explainability

      • Individual Conditional Expectation (ICE)
      • Accumulated Local Effects (ALE)
      • Example-based explanations
    • Fairness metrics
      • Disparate Impact Ratio

      • Demographic Parity
      • Treatment Equality
